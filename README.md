# Amazon Best Sellers 2025 - Exploratory Data Analysis

![GitHub Repo Size](https://img.shields.io/badge/Status-Active-brightgreen)

## Overview
This repository contains an **exploratory data analysis (EDA)** of Amazon's top-selling products in 2025. Using Python libraries like **Pandas, NumPy, Matplotlib, and Seaborn**, this project uncovers insights about product categories, pricing trends, customer ratings, and more.

---

## Dataset
- **Source:** Kaggle  
- **Dataset Name:** Amazon Best Sellers 2025  
- **Description:** Contains details of Amazon's best-selling products including category, price, ratings, reviews, and release date.

---

## Project Goals
- Understand distribution of products across different categories.
- Analyze price trends and average ratings.
- Visualize relationships between product features using charts and heatmaps.
- Practice data cleaning, manipulation, and visualization using Python.
